# Model klasörü

Eğittiğiniz YOLOv8 ağırlık dosyanızı buraya `best.pt` adıyla koyun:

```
backend/models/best.pt
```

Farklı bir konum kullanmak isterseniz `.env` içindeki `MODEL_PATH`
değişkenini güncelleyin.

Modelin 6 sınıfı sırasıyla:
1. Fish
2. Human Diver
3. Robot
4. Stingray
5. Turtle
6. Wrecks - Ruins
